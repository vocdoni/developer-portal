VocdoniSDKClient

@vocdoni/sdk / VocdoniSDKClient

Class: VocdoniSDKClient

Main Vocdoni client object. It's a wrapper for all the methods in api, core and types, allowing you to easily use the vocdoni API from a single entry point.
