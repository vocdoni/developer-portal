Dot notation to object conversion. Takes any object as first argument and uses the string dot notation from the<br />second argument (i.e. 'a.child.node') to access that given object value.

**Returns**: any
- Return the object data

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|obj|any|||Object to be accessed by dot notation|
|dot|string|||Dot notation string to extract object data|
