# Layout Components

Several design components are provided by the ui-components library to help with election formatting.

## Button

```jsx live
<Button>
Click Me!
</Button>
```

## Horizonal Rule

```jsx live
<HR />
```

## Image

Displays an image from the given source URL. This component is unique in that it can display images directly from and [IPFS](https://ipfs.io) URI. 
```jsx live
<Image src={'https://source.unsplash.com/random'}/>
```

## Markdown 

```jsx live
<Markdown >
{
`# Markdown text example
- Full featured
1. **So useful**`
}
</Markdown>
```
